# Design Portfolio

A professional design portfolio website inspired by phantom.land, featuring fluid motion, bespoke layouts, and an infinite grid component.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Motion.dev
- Lucide React
- React Three Fiber
- shadcn/ui
